## 프로젝트 소개

순수 자바스크립트 (타입스크립트)를 이용하여 TodoList를 구현합니다.

### 사용 기술

이 프로젝트는 다음과 같은 기술들을 사용합니다:

- Typescript
- Webpack
- Babel
- Jest

### 실행 커멘드

- Dev
  ```sh
  npm install
  npm run serve
  ```
- Prodction
  ```sh
  npm run build
  ```
- Test
  ```sh
  npm run test
  ```
